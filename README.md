# 📈 Yes Bank Stock Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts the **closing stock price of Yes Bank** using historical stock market data and multiple Machine Learning regression algorithms.

The project includes complete data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and comparison of different regression models.

---

## 🎯 Objective

To build a regression model capable of accurately predicting the closing stock price of Yes Bank based on historical stock data.

## 📊 Dataset

The dataset contains historical stock prices of Yes Bank including:

- Date
- Open Price
- High Price
- Low Price
- Close Price

**Note:**  
The dataset is **not included** in this repository.

You can download it from Kaggle and place it in the project folder before running the notebooks.
